Simple Selectors In CSS

the first selector is a element selector which can be selected by typing in the element tag followed by the curly braces


the next selector is an ID selector which is selected by typing a(#) followed by curly braces and the name of the ID in question


the last selector is a class selector which is selected by typing a (.) followed by the curly braces again and the name of the class

Other Selectors

another selector is a descendant selector where you are trying to reach a specific child element inside a specific parent element note that there is no comma between the two tags just white space


the next selector is the child selector which looks for elements that are a direct child element of the parent element you can show this by putting a greater than sign before the child element


a cool selector is the attribute selector which will look for an element with a specific attribute for example you could select all images with the attribute of (alt) snapshot


finally the last selector is the pseudo selector for example on an anchor tag (a) when he link is clicked it will have a pseudo class of visited you could change the color of the link depending on whether the user has clicked on the link or not

The Big Three Box Properties


The big three box properties are as following

Padding
padding adds space between the border of a box and the content of the box

Border
Border around the box

Margin
Margins is the space around the border which pushes the box away from other boxes
Vertical Margins Collapse

when applying margins to a either a top or a bottom for example 1 box had a bottom margin of 5px and the other box had a top margin of 5px and they where on top of each other the total space between them would not be 10px but would be 5px as the margins would overlap on top of each other

this rule only applies to vertical margins and does not apply to horizontal margins

Positioning In CSS
Static Positioning
Static Positioning stacks the elements one after the other down the page and is the default

Relative Positioning
Relative positioning moves the element from its default location making it easy to manipulate specific elements to get them where you want in the webpage

Absolute Positioning
Absolute positioning moves the element to a desired location accurately relative to the body element
